# matlab-image-filter
A image filter program based on matlab, including sketch, oil painting, cameo, mosaic, snow, facula, asthenia and coloring.

程序运行说明

主程序命名为main.m，打开运行即可。其调用的函数程序分别为sketch（素描算法）、oil painting（油画算法）、cameo（浮雕算法）、mosaic（马赛克算法）、snow（大雪滤镜）、facula（光斑滤镜）、asthenia（背景虚化算法）、coloring（色调滤镜），以及其他功能实现的函数，包括rgb2hsi（RGB色彩空间转HSI色彩空间算法）和hsi2rgb（HSI色彩空间转RGB色彩空间算法）

运行时程序会对图像’test1.jpg’’test2.jpg’’test3.jpg’进行处理，处理的结果用’sketch.jpg’、’oilpainting1.jpg’、’oilpainting2.jpg’、’cameo.jpg’、’mosaic.jpg’、’snow.jpg’、’facula.jpg’、’asthenia.jpg’、’bright.jpg’、’blue_tune.jpg’命名，表示素描，油画等的处理结果。
